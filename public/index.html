How to customize Flash?

Costomization of flashcss means making changes to the source and rebuilding it before using in your website

1 Create a new *.scss file 2 Override variables in /src/partials/*.scss 3 Include src/flash.scss last before rebuilding by running npm run pack

Build files will be written to the build directory

What's in the box?

Buttons
Components

Flash comes with predefined classes for creating buttons. The following theme buttons are available by default

Usage
<button class="flash-button flash-primary-button"> Primary Button</button>
<button class="flash-button flash-secondary-button"> Secondary Button</button>
<button class="flash-button flash-info-button"> Info Button</button>
<button class="flash-button flash-warning-button"> Warning Button</button>
Customize

You can customize the color theme by assigning new values to the variables defined in src/_buttons.scss

Navbars

Flash provides utility classes for placing navigation bar itams on different positions. We currently have support for the 3 main positions below

Left Navbar

Right Navbar

Center Navbar
Navbar utility classes

Use the following classes to create positioned navbars.

.navbar declares a navigation bar


.navbar-item declares a navigation bar item

Theme specific classes

Use .flash-bg-primary, .flash-bg-secondary, flash-bg-info, .flash-bg-warning in conjunction with .navbar to give the navbar the corresponding theme colors.

Please checkout the source for more classes that work with the Navbar component

Code
<nav class="navbar navbar-left flash-bg-primary">
.. Navbar items ..
< /nav>

<nav class="navbar navbar-right flash-bg-secondary">
.. Navbar items ..
< /nav>

<nav class="navbar navbar-center flash-bg-info">
.. Navbar items ..
< /nav>
Grid System
Component

Flash Ships with a 6 column responsive grid system.

Usage

Create a flash container to contain the grid columns

.flash-container

Add columns to the flash container using one of the defined column configurations

.column-1-6, .column-2-6, column-3-6, column-4-6, column-5-6, column-6-6

You can also use the shorthand classes to create the columns

.column-1, .column-2, .column-3, .column-4, .column-5, .column-6

Section
Description

Section is a container component used to create aligned columns that follow the container layout

Use .flash-section to create a page section that occupies 100% of its container.

Classes

.flash-section-primary: Creates section with primary theme background

.flash-section-secondary : Creates section with secondary theme background

.flash-section-large: Creates a larger section container

Input classes

Use the following classes with form inputs

Available classes

Use .flash-form or .flash-block-form on a form element to create a flash block form

Use .flash-form-inline to create an inline display form

Use .flash-bare-input on input items to create bare form input form fields as seen above

Extending

Animations

Flash provides animations that add a better user experience to our components. Animations are defined in classes and all animation classes follow the pattern @-[animation-type].

Flash animations can be applied to any element to animate it.

Available animations
@-flash-out
@-flash-in
@-slide-left
@-slide-in-top
@-slide-in-bottom
Extending

You can also modify the animation behaviour by changing he duration in src/_utils.scss
